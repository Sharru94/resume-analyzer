# AI-Ready Resume Analyzer

## Overview
This is a Python-based Resume Analyzer that helps job seekers optimize their resumes by comparing them against job descriptions.  
The tool calculates a **skill match percentage**, identifies missing skills, and generates a basic report for improving resume content.

---

## Features
- Reads a resume and job description from text files
- Identifies matched and missing skills
- Calculates skill match percentage
- Generates a text report
- Simple, beginner-friendly, and offline

---

## How it Works
1. Store your **resume skills** in `resume.txt`
2. Store the **job description skills** in `job_description.txt`
3. Run the Python script:
```bash
python analyzer.py
